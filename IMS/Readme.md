## Airtel Inventory Management System(IMS)

A simple and reliable system for managing Airtel devices such as laptops and desktops.
It allows tracking of assets, assignments, returns, and audit history using a local MySQL database.

## Features

* Add and manage assets (laptops, desktops, tablets)
* Assign assets to employees
* Return assigned assets
* Track asset status (Available / Assigned)
* Audit logging for all actions
* Simple dashboard for monitoring

## Technologies Used

* Java (Spring Boot)
* MySQL Database
* Thymeleaf (HTML Templates)
* Hibernate / JPA
* HTML, CSS

## Database Setup

1. Open XAMPP / MySQL Workbench / phpMyAdmin
2. Import the file inventory\_db.sql
3. Or allow Spring Boot to auto-create tables using Hibernate
